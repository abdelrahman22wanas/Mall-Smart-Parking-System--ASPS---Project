# Parking Sensor System (Verilog)

This repository contains the Verilog source files for the parking sensor system project.

## Files

- `asps_tb.v` - testbench
- `asps_top.v` - top-level module
- `cost_calculator.v` - parking cost logic
- `debounce_edge.v` - button/signal debounce and edge detect
- `entry_fsm.v` - entry finite state machine
- `exit_fsm.v` - exit finite state machine
- `global_clock.v` - clock generation/division
- `seven_seg_encoder.v` - seven-segment display encoder
- `timestamp_buffer.v` - timestamp storage/buffer
- `updown_counter.v` - vehicle count up/down logic
